# slack-reaction

## Overview
Aggregate slack's reaction.

![sample](https://imgur.com/lPDCbFG.jpg)

## Install
```bash
$ git clone https://github.com/yusukeito58/slack-reaction.git
$ cd slack-reaction
$ npm install
$ mv .env.sample .env
$ vi .env
# modify parameter
$ node main.js
```
